Supported Algorithms

Choose from industry-standard cryptographic hash functions for your security needs

Legacy

MD5

128-bit

Fast but cryptographically broken. Use only for checksums, not security.

Legacy

SHA-1

160-bit

Deprecated for security. Collision attacks demonstrated in 2017.

Recommended

SHA-256

256-bit

Industry standard. Used in Bitcoin, SSL certificates, and most security applications.

Secure

SHA-512

512-bit

Maximum security for sensitive applications. Slower but more resistant to attacks.

Modern

BLAKE2s

256-bit

Faster than MD5, more secure than SHA-2. Optimized for 8-32 bit platforms.

Modern

BLAKE2b

512-bit

Fastest secure hash function. Used in Argon2, WireGuard, and modern crypto.
